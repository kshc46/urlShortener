URL Shortener

Hello human, I am a simple app that holds all the information in the world. Just kidding, I only shorten URLS. Feed me an address tagged at the end of my URL and I'll give you a super nifty short code back so you can access your destination without typing so damn much. Like so: https://short-url-kshc46.c9users.io/new/https://www.google.com... this returns 8814. So now you only have to type https://short-url-kshc46.c9users.io/8814. Wow!
